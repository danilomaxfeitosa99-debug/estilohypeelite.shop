# Hera Yacht Charter - Versão CNPJ 65.042.853/0001-31

Esta pasta contém uma variação do site institucional da **Hera Yacht Charter**, configurada para uso com o CNPJ **65.042.853/0001-31** e textos ligeiramente ajustados em relação à versão base, mantendo a mesma identidade visual náutica.

## Visão Geral

O site apresenta os serviços de charter de iates, experiências a bordo e frota selecionada para navegações na Costa Verde e litoral paulista.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, luxuoso e náutico (azul marinho, vermelho e bege).
-   **Objetivo:** Servir como landing page premium para campanhas de charter e presença institucional da marca.
